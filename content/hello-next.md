# 快速开始

本项目示例展示如何使用 **Next.js** 静态导出并托管到 GitHub Pages。

## 步骤

1. 编写 Markdown 教程内容。
2. 使用 `next build && next export` 生成静态站点。
3. 提交到 `gh-pages` 分支并开启 Pages。

```bash
npm run build
```

完成！
